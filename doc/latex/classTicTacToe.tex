\doxysubsection{Detailed Description}
Класс игры в крестики нолики. Класс написан в соотвествии с паттерном одиночка, так как в процессе игры должен существовать только один экземпляр класса. С созданием экземпляра класса начинается игра. При уничтожении экземпляра игра заканчивается.
